Heroes

Here we consume the Marvel API to retreive Earth's mighties heroes... and villians. The data is then presented in a table with standard pagination, sorting and searching. You may also select heroes and add them to a team.

Star Wars

This page features infinite-scrolling that we use to continuously consume the Star Wars API. It also conveniently remembers which character was selected when you view thier details.

These list allow you to drag-and-drop an item from one list to another. I must have been using Jira and thought, "Hey, let me try that!"

Happy Trails

This page uses the Google Cloud Platform and its mapping API to display some of my favorite hiking trails around Littleton.

05

Change Orders

The left-side displays all items in an infinite-scrollable list. Items can then be selected and their details displayed on the right-side. Try the multi-select checkbox for some interesting results.

06

Themes & Settings

A theme is the set of colors that will be applied to the Angular Material components. You can also choose a route-transition. These transitions are then used when switching routes.

07

Large Tables

This shows a way to handle large tabular data. The table's header and first column are fixed in place; that is they don't scroll like the other columns do. The user may also choose which columns to include/exclude.

08

Recursion

This example uses recursion. There is an unknown number of line-items. Each line-item can be nested to the nth-level. So we use recursion to create a nested hierarchy of line-items and all their children.

09

Tree & Stuff

This is a hierarchical-tree-view that allows users to select items. The tree can have nth-level of children. Once selected the item is brought over into a grid. The heirarchy of parent/child/grand-children is maintained.
